Codex 优化一键脚本（幂等 — 已完成项自动跳过）
功能: 关闭完整性校验 + 插件解锁 + DNS加速 + 中文语言

不依赖第三方插件开启，0依赖，纯脚本开启，官方插件功能。


1.安装codex Macos Interl 官方原版
2.使用终端运行 codex-optimize-V2.sh 自动化脚本
3.启动 codex.app，进入设置项中更改 中文 语言，可以使用 第三方 模型 API_KEY 登录。
最好有魔法（因为设置中的 中文 语言 需要魔法），之后 重启一次 codex.app，看看是否转换成功
4.转换成中文语言后，再次终端运行 Codex-DNS-wrapper-v2.sh 这个脚本
这个脚本为了启动codex.app时，不会因为联网openai相关的网络，而导致启动加载慢（非常的慢，因机而言，本机需要5分钟）
这步为了跳过openai网络检查。
